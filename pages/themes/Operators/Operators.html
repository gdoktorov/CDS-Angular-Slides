Аритметични оператори (Aritmetic operators)

  • Събиране: '+' 
    
          console.log(2 + 4); //6
          console.log(2 + -4); //-2
  • Изваждане: '-' 
    
          console.log(20 - 4); //16
          console.log(20 - -4); //24
  • Умножаване: '*' 
    
          console.log(2 * 4); //8
          console.log(2 * -4); //-8
  • Делене: '/' 
    
          console.log(100 / 5); //20
          console.log(100 / -5); //-20
          console.log(7 / 2); //3.5

Аритметични оператори (Aritmetic operators)

  • Делене по Модул (остатък): '%'

    Връща остатъка от целочислено делене

    
          console.log(3%1); //0
          console.log(3%2); //1
          console.log(3%3); //0
          console.log(3%4); //3
  • Степенуване: '**'

    Въвежда се от 2016г. в ES7 и работи само с най-новите версии на браузерите 

    
          console.log(2**10); //1024

    За по-стари браузери използваме Math.power()

    
            console.log(Math.pow(2,10)); //1024

Аритметични оператори (Aritmetic operators)

  • Инкрементиране: '++' 
    
          var i = 2;

          //postfix increment:
          var j = i++; // j = 2, i = 3

          //prefix increment:
          var j = ++i; // j = 3, i = 3
  • Декрементиране: '--' 
    
          var i = 2;

          //postfix decrement:
          var j = i--; // j = 2, i = 1

          //prefix decrement:
          var j = --i; // j = 1, i = 1

Increment - examples


      var x = 1;
      console.log(x++); //1
      console.log(x);   //2

      var y = 1;
      console.log(++y); //2
      console.log(y);   //2

Increment - examples


        var x = 1;
        var y = 1;

        console.log( x++ + ++y); // 3

        console.log(`x = ${x}`); // x = 2
        console.log(`y = ${y}`); // y = 2

За по-сложни математически операции, може да използвате методите на вградения в езика Math object 


      Math.round(3.65); //4
      Math.pow(2,10);   // 1024
      Math.min(1,2,3,4,5); // 1
      Math.max(1,2,3,4,5); // 5

Оператор за присвояване (Assignment operator)

Assignment '=' 

          LHS = RHS;
LHS: Left Hand Side: трябва да бъде декларирана променлива. В противен случай ще се създаде глобална променлива, което е изключително лоша практика.
RHS: Right Hand Side: израз, който се изчислява и стойността му се записва в LHS променливата

          // декларираме променливите x и y:
          var x, y;

          // записваме в паметта, сочена от x, числото 3:
          x = 3;

          // в y записваме стойността на x:
          y = x;

          // в x записваме стойността на x, повдигната на степен 2:
          x = x ** 2; // след този израз в x имаме стойността 9

Оператор за присвояване (Assignment operator)

Операцията "присвояване" връща стойност! И това е стойноста, която ще се запише в LHS променливата
Поради това, можем да имаме няколко оператора за присвояване в израз:

      // декларираме променливите x и y
      let x;
      let y;

      // присвояваме и на двете стойността 1
      y = x = 1;

      console.log(`x = ${x}`); // x = 1
      console.log(`y = ${y}`); // y = 1

Оператор за присвояване (Assignment operator)

Понятията "LHS" и "RHS" се използват не само в контекста на assignment оператора. Най-общо, когато променлива се използва като "LHS", то тогава записваме стойност в нея. Когато се използва като "RHS", то тогава четем стойност от нея.

Оператор за присвояване (Assignment operator)

  • Съкратен запис на присвояване + аритметична операция 
    
          var x = 4;
          x += 3; // x = x + 3;
          x ­-= 3; // x = x – 3;
          x *= 3; // x = x * 3;
          x /= 3; // x = x / 3;
          x %= 3; // x = x % 3;

Слепване на низове (String concatenation)

  • Конкатенация (слепване) на низове: '+' 
    
          var firstName = "Ada";
          var surName = "Byron";
          console.log (firstName + " " + surName); // Ada Byron
  • Ако от двете страни на '+' имаме различни типове, например number и string, то JavaScript преобразува числото до string и извършва concatenation, а не събиране! 
    
          console.log(2+"1"); // 21
  • Преобразуването на типове се нарича type coercion.

Оператори за сравнение - примери


      console.log(2 == 2);    // true
      console.log(2 == "2");  // true (the string is coerced to a number value.)
      console.log(2 === "2"); // false (no type coercion)

      // the Boolean operand "true" is converted to 1; "false" is converted to 0
      console.log(2 == true);   // false
      console.log(2 > true);    // true
      console.log(true > false);  // true
      console.log(-24 == true );  // false

      // lexicographic sorting, when both operands are strings
      console.log( "2" > "19999" )  // true, as ("2" > "1") is true
      console.log( "b" > "azzzz" )  // true, as ("b" > "a") is true
Забележете, че всички оператори за сравнение винаги връщат Булев тип стойност (true/false)

логически оператори (Logical Operators)

Обикновено, логическите оператори се използват над Булев тип операнди, за да съставим по-сложни изрази.
Например: "Потребителят е пълнолетен и е от България"
Но логическите оператори могат да се използват и над друг тип данни. В такъв случай, те не връщат Булева стойност.

логически оператори - примери


      console.log( !true ); // false
      console.log( true || false ); // true
      console.log( true && false ); // false

      console.log( 2>1 && "a">"b" ); // false ( true && false = false)

Изчисляване на истинностни стойности

Логическите оператори в JS могат да работят върху различни типове данни. За да се изчисли резултата, за всеки един от операндите трябва да се изчисли истинностната му стойност (true/false).
В сила са следните правила:

Type: =False =True
number 0/NaN всяко число, различно от 0/NaN
string "" всеки низ, който не е празен
undefined undefined -
null null -
