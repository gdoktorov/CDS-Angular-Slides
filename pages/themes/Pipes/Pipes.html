Transform data with pipes, not filters!

Transform data with pipes, not filters!

AngularJS (Angular 1.X) - uses filters to transform data
In Angular (Angular2+) the filters are now called pipes
Most of AngularJS filters have their Angular pipes equivalents
There is no equivalent for filtering or sorting lists No FilterPipe or OrderByPipe @angular docs

How to use Pipes

How to use Pipes

A pipe takes in data as input and transforms it to a desired output
Angular includes several pipes to add to your template.
There is no need to import them or add them as directives or anything—just start using them.
A pipe is denoted with the pipe symbol : |

Syntax


      {{ data | pipe }}

Built-in Pipes

Built-in Pipes

lowercase

Transforms a string into a lower-case string


      {{ 'string' | lowercase }}
    

      
Lowercase: {{'Ada Byron' | lowercase}}

    

      Lowercase: ada byron

uppercase

Transforms a string into a upper-case string


      {{ 'string' | uppercase }}
    

      
Uppercase: {{'Ada Byron' | uppercase}}

    

      Uppercase: ADA BYRON

currency

Adds currency info to a number


      {{ value | currency [ : currencyCode [ : display [ : digitsInfo [ : locale ] ] ] ] }}
    

      
Currency: {{ 23.55 | currency }}

    

      Currency: $23.55

More on CurrencyPipe @angular docs

percent

Transforms a number into a percent


      {{ 'string' | percent }}
    

      
Percentage: {{ 0.5 | percent }}

    

      Percentage: 50%

date

Transforms a datetime object or string representing a date into a date formated string
DatePipe @angular docs

      {{ value | date [ : format [ : timezone [ : locale ] ] ] }}
    

      
Date: {{ '2018-01-15T09:03:01+0900' | date }}

      
Date: {{ '2018-01-15T09:03:01+0900' | date:'y MMMM - EEEE d' }}

    

      Date: Jan 15, 2018

      Date: 2018 January - Monday 15

more built-in pipes demos


      import { Component, OnInit } from '@angular/core';

      @Component({
        selector: 'app-built-in-pipes',
        template: `
          
{{userName}} piped into 'uppercase' is: {{userName | uppercase}}

          
Number {{number}} piped into 'percent' is: {{number | percent}}

          
Current date piped into 'date:'dd.MM.y' is: {{ now | date:'dd.MM.y' }}

          
Current time piped into date:'HH:mm:ss' is: {{ now | date:'HH:mm:ss' }}

        `,
        styles: [
        ]
      })
      export class BuiltInPipesComponent {
        userName = 'ada';
        number = 0.2;
        now = new Date();
      }

Custom Pipes

Custom Pipes

Create custom pipes

You can use the CLI:

        ng generate pipe pipeName
Or, you can create your own class with @Pipe annotation and:
the class must implement the PipeTransform interface
imports the Pipe and PipeTransform

example


      import { Pipe, PipeTransform } from '@angular/core';

      @Pipe({
        name: 'fancyQuotes', pure:false
      })
      export class FancyQuotesPipe implements PipeTransform {
        transform(value: any): string {
          return `«${value}»`;
        }
      }
    

      template: `
        

          {{'ada'|fancyQuotes}}
        

      `,
